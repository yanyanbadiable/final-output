JavaScript Declaring Variables




4 ways to Declare a JavaScript variables:


1. Using var
2. Using let
3. Using const
4. Using nothing


What are Variables?


In JavaScript, a variable is a container that can hold a value. It is declared using the var, let, or const keyword, followed by the variable name.


For Example: 

     
    Using var:           Using let: 
      var x = 5;           let x = 5;
      var y = 6;           let y = 6;
      var z = x + y;       let z = x + y;
           
    Using const:         Using variable 
      const x = 5;       without declaring: 
      const y = 6;          x = 5;
      const z = x + y;      y = 6;
                            z = x + y;

When to use Javascript var ?


Variables declared with var have function-level scope, meaning that they are accessible within the function they are declared in, or globally if declared outside a function. var variables can also be re-assigned and hoisted to the top of their scope, which can lead to unexpected behavior if not used carefully.


When to use Javascript let ?


Variables declared with let have block-level scope, meaning that they are only accessible within the block they are declared in (for example, within a loop or an if statement). let variables can be re-assigned but cannot be re-declared within the same block.


When to use Javascript const ?


Variables declared with const are also block-scoped and cannot be re-assigned once they are initialized. This means that the value of a const variable cannot be changed after it is set, although the properties of an object or elements of an array assigned to a const variable can still be modified.
