JavaScript Arithmetic Operators




What is JavaScript Arithmetic Operators?


Arithmetic operators perform arithmetic on numbers (literals or variables).


     
    Operator       Description

       +           Addition
       -           Subtraction 
       *           Multiplication     
       /           Division 
       **          Exponentation 
       %           Modulos (Remainder) 
       ++          Increment 
       --          Decrement

JavaScript Arithmetic Operation


1. Addition


The addition operator is used to add two or more numbers together.


For Example: 


     
    let x = 5;
    let y = 2;
    let z = x + y;

2. Subtraction


The subtraction operator is used to subtract two or more numbers together.


For Example: 


     
    let x = 5;
    let y = 2;
    let z = x - y;

3. Multiplication


The multiplication operator is used to multiply two or more numbers.


For Example: 


     
    let x = 5;
    let y = 2;
    let z = x * y;

4. Division


The division operator is used to divide two or more numbers.


For Example: 


     
    let x = 5;
    let y = 2;
    let z = x / y;

5. Exponentation


The exponentation operator raises the first operand to the power of the second operand.


For Example: 


     
    let x = 5;
    let y = 2;
    let z = x ** y;

6. Modulos (Remainder)


The modulo operators returns the division remainder.


For Example: 


     
    let x = 5;
    let y = 2;
    let z = x % y;

7. Increment


The increment operator increments numbers.


For Example: 


     
    let x = 10;
    x++;
    let z = x;

8. Decrement


The decrement operator decrements numbers.


For Example: 


     
    let x = 10;
    x--;
    let z = x;
