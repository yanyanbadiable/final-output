JavaScript Pop-up Messages




What is JavaScript Pop-up Message?


JavaScript pop-up messages are commonly used to display alerts, confirmations, and prompts to the user.



Here are three types of Pop-up Messages that can be created using JavaScript:


1. Alert


An alert box is used to display a message to the user. It is a simple pop-up window with an "OK" button that the user can click to dismiss the message.


For Example: 

     
        alert("Hello, world!");

2. Confirm


A confirm box is used to ask the user to confirm or cancel an action. It is a pop-up window with "OK" and "Cancel" buttons.


For Example: 

     
    var result = confirm("Are you sure you want to delete this item?");    
    if (result) {
    alert("The item has been deleted !!!");
    } else {
    alert("Okay, you dont delete the item !")
    }

3. Prompt


A prompt box is used to ask the user to enter some input. It is a pop-up window with a text input field and "OK" and "Cancel" buttons.


For Example: 

     
    var name = prompt("Please enter your name:");
        if (name) {
            alert("Hello, " + name + "!");
        } else {
            alert("You didn't enter your name.");
        }
